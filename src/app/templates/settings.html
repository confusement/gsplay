GsPlay Editor : Online Shader Editor

What is this
Guide - start with presets and play with the code, for proper guide read/watch Inigo Quilez's tutorial and check out his website https://www.shadertoy.com/
Changes stored in local storage in browser, which is not not sent to any server.
Pause button does not freeze time, it stops continous render, i.e. renders one frame after a change is made
Auto Compile set to 1 sec intervals
Shader toy naming conventions used, refactoring ability may be added
glhf

Shortcuts -
Ctrl + E         - Show/Hide Editor
Alt + arrows - Switch
`                      - Open Settings
Ctrl + Space - Pause Play
